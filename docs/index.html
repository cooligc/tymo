Schedule It

A lightweight Java scheduling framework with custom annotations and high availability support

View on GitHub Get Started

🚀 Simple Annotations

Use @ScheduledTask and @Job annotations to define scheduled tasks with minimal code.

🔒 High Availability

Database-based locking ensures tasks run on only one instance in distributed environments.

🧵 Thread Pools

Flexible thread pool management with custom pool names and configurations.

📦 Executable JAR

Build and deploy as a standalone executable JAR with all dependencies included.

Quick Start

1. Add to your project

<dependency> <groupId>io.cooligc</groupId> <artifactId>schedule-it</artifactId> <version>1.0-SNAPSHOT</version> </dependency>

2. Create a scheduled task

@Job public class MyTasks { @ScheduledTask(fixedDelay = 5000) public void myTask() { System.out.println("Task executed!"); } }

3. Run the scheduler

SchedulerService scheduler = new SchedulerService(); scheduler.start(); scheduler.registerJobs();
